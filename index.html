Welcome Commander

Using custom names is one of the most thrilling aspects of the various XCOM names. But wouldn't it be even better if you could draft custom names at random instead of having to rename soldiers manually?

This is why XCOM Namelist Generator exists - it allows you to generate custom namelist files for your XCOM games for random drafting of names you've picked in advance.

XCOM Namelist Generator currently works with XCOM: Enemy Unknown, XCOM: Enemy Within, XCOM: Long War and OpenXcom. For anyone hoping for Xenonauts support, consider gifting the game to me as I don't really have interest in it otherwise. My Steam account is here.

How to use the namefiles

Make sure to back up the original namefiles first!

For Enemy Unknown and Enemy Within, the namelist is a file called DefaultNameList.ini.

For Enemy Unknown, you should place this file into [XCOM Install Path]\XComGame\Config\

For Enemy Within & Long War, the path is [XCOM Install Path]\XEW\XComGame\Config\

In OpenXcom, the namelist is a bunch of Language.nam files in a folder called SoldierName.

You should extract the downloaded archive into [OpenXcom Path]\data\
